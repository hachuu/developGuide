# 간단한 RAG OpenAI Web App
## 설명
- 이 웹앱은 OpenAI의 GPT 모델을 사용하여 질문에 대한 답변을 제공하는 간단한 RAG(Reading, Answering, Generating) 시스템입니다.
- 사용자는 질문을 입력하고, 웹앱은 해당 질문에 대한 답변을 생성합니다.
