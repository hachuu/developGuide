# 1-1. 강좌 소개와 맛보기
1. button 제어 방식

```html
<head>
  ...
  <script src="https://cdn.jsdelivr.net/npm/vue@2.7.13/dist/vue.js"></script>
</head>
<body>
  <div id="root">
    <div>좋아요 눌렀음</div>
    <button>Like</button>
  </div>
</body>
<script>
  const app = new Vue({
    el: #root,
    data: {
      liked: false
    },
    methods: {
      onClickButton() {
        this.liked = true; // this는 data를 나타냄
      }
    }
  })
</script>
```

2. 정리 요약
-  methods에서 this는 data를 나타냄
-  data를 업데이트 시키면 dom의 element가 알아서
